# Business-Insights-360-AtliQ-Hardware
🔷 Project Overview

Business Insights 360 is an end-to-end Business Intelligence solution developed using Power BI to provide a 360° view of business performance.

The project focuses on transforming raw data into actionable insights across key business functions such as Finance, Sales, Marketing, Supply Chain, and Executive view enabling data-driven decision-making.

🔷 Problem Statement

AtliQ Hardware faced challenges in managing and analyzing large volumes of data using Excel, resulting in fragmented reporting, limited visibility, and slower decision-making.

🔷 Solution

Developed an interactive Power BI dashboard by integrating multi-source data and designing a structured data model (star schema).

Implemented DAX-based KPIs to track key metrics like revenue, profit margins, and performance across regions and channels.


🔷Dashboard Views

🏠 Home Page – Central navigation hub

💰 Finance View – Revenue & profitability analysis

📈 Sales View – Customer & product performance

📊 Marketing View – Segment & market insights

🚚 Supply Chain View – Forecast accuracy & inventory tracking

🧭 Executive View – High-level KPIs for decision-making

🔷 Key Features

Interactive dashboards with filters & slicers

Drill-down and drill-through capabilities

Dynamic KPI tracking using DAX

Trend analysis (monthly & yearly)

Region and category-level insights

🔷 Tools & Technologies

Power BI

Power Query

DAX

SQL

Excel


🔷 Key Insights

Identified top-performing regions and products

Highlighted underperforming segments

Analyzed profit margin trends

Enabled comparative performance analysis

🔷 Business Impact

Improved visibility across business functions

Reduced manual reporting effort

Enabled faster and more accurate data-driven decisions
